--- Conteúdo da Disciplina ---

1. Arquitetura da Web: conceitos básicos sobre redes de computadores,
arquiteturas de aplicações cliente-servidor, o papel do navegador web,
conceitos e exemplos de servidores web (http e de aplicação);

2. Protocolos de comunicação da internet: conceitos sobre protocolos de
comunicação importantes para o desenvolvimento web (TCP/UPD, HTTP, FTP,
SMTP e DNS);

3. Internet e web: introdução aos conceitos de aplicações web, infraestrutura
web, infraestrutura em nuvem, serviços web, URLs e domínos;

4. Conceitos básicos de programação de computadores: programação
imperativa, programação orientada a eventos, programação
síncrona/assíncrona, ferramentas básicas de programação, versionamento
de código (git);

5. Fundamentos da linguagem HTML: conceito de linguagem de marcação de
texto, estrutura do documento HTML, elementos de layout, elementos de
layout não-semânticos, elementos de destaque de texto, elementos de
navegação, mídia, formulários e tabelas;

6. Fundamentos da linguagem CSS: seletores, herança, box model, background,
bordas, cores, manipulação de texto, overflow de conteúdo, valores,
unidades de medida e manipulação de imagens;

7. Fundamentos da linguagem Javascript: gramática básica, introdução aos
tipos de dados e funções, expressões e operadores;

8. Introdução a Bancos de Dados para desenvolvimento web: conceito de
bancos de dados, bancos de dados relacionais, bancos de dados NoSQL e
mapeamento objeto-relacional;

9. Desenvolvimento de páginas para internet: ferramentas de programação
HTML, CSS e JavaScript, ferramentas para análise de documentos HTML e CSS, análise de sites existentes,

10. Conceitos básicos sobre interfaces de usuário dinamicamente adaptáveis:
layout baseado em proporções, layout responsivo, formatação CSS condicional com media queries.

11. Frameworks para desenvolvimento fullstack: conceitos e exemplos de frameworks para desenvolvimento integral, ou parcial, de aplicações web em diferentes linguagens de programação

12. Aplicações web com renderização no servidor: utilização de templates html e conceitos de arquitetura de sistema Model-View-Controller,

13. Desenvolvendo APIs web: conceito de interface de programação de aplicações, arquitetura de serviços e micro-serviços web, protocolos de aplicação REST e SOAP, segurança, formatos de dados, e frameworks para
desenvolvimento de APIs.

14. Consumindo e enviando dados de/para APIs web: ferramentas de consulta
HTTP, consultas com segurança, consumindo/enviando dados de aplicações
web.


--- Relação entre as páginas de referência e o conteúdo da disciplina ---

1 - Para a Arquitetura da Web será utilizado a ferramenta de http para criar a base da aplicação web;

2 - Para desenvolvermos os protocolos de comunicação iremos usar o ambiente de desenvolvimento chamado de VSCode, esse será usado para a maioria das codificações;

3 - Internet e Web são conceitos diferentes, nos quais são de fácil confusão, então para isso podemos usar para pesquisar seus diferentes conceitos o navegador, seja Chrome, Opera, Firefox e etc, ferramentas como ChatGPT podem ser usados também, porém tomar cuidado com a veracidade das informações;

4 - Conceitos básicos de programação poderão ser pesquisados da mesma forma dos conceitos de aplicações web;

5 - Os fundamentos da linguagem de HTML serão estudados pela documentação da linguagem evindenciada no começo deste documento e será usada para a criação dos sites;

6 - O CSS é diferente do HTML porque enquanto o html é relacionado à estrutura do site, como parágrafos, imagem, videos e etc, o CSS é aplicado na parte visual do projeto, como cores, fontes ou qualquer outro valor estético;

7 - JavaScript serve para a implementação de recursos mais complexos em páginas web, como por exemplo: mapas interativos, gráficos 2D/3D e etc;

8 - O banco de dados será utilizado para guardarmos nossos documentos e podermos acessá-los das nossas aplicações, seja preço de produtos, nome e qualquer outra informação que seja importante na visualização;

9 - HMTL, JavaScript e CSS, serão usados simultaneamente para a criação de uma página web;

10 - Interfaces dinâmicas serão implementadas usando CSS através de programas de edição de texto;

11 - Os Frameworks serão selecionados e usados através das linguagens de programação definidas ao projeto;

12/13/14 - Toda a parte de desenvolvimento da aplicação web propriamente dita.
